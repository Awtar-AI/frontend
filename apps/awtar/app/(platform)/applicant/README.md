# Awtar AI Applicant Portal - Quick Navigation Guide

This project is a modern, AI-powered recruitment platform for applicants. Below are the key routes to explore the features implemented so far.

## 🚀 Key Routes
- **Dashboard:** `http://localhost:3000/applicant/dashboard` (Overview, recommendations, saved jobs)
- **Job Search:** `http://localhost:3000/applicant/jobs` (Explore, filter, and discover roles)
- **Job Details:** `http://localhost:3000/applicant/jobs/[id]` (Detailed view of a job posting)
- **Application Flow:** `http://localhost:3000/applicant/apply/[id]` (Multi-step application process)
- **Profile:** `http://localhost:3000/applicant/profile/public` (Applicant public landing page)
- **Edit Profile:** `http://localhost:3000/applicant/profile/edit` (AI resume parsing and profile management)
- **Saved Jobs:** `http://localhost:3000/applicant/saved` (Manage bookmarked opportunities)

## 🛠️ Developed Features
- **AI Match Analysis:** Dynamic matching indicators on job cards.
- **Smart Filters:** Categorized job discovery by experience, role, and type.
- **Modern UI:** Premium glassmorphism aesthetics and micro-animations throughout.
- **Clean Code:** 100% Biome linting compliance for high reliability.
